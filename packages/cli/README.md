# XCHEME

This package provides the XCHEME CLI.

## Get started

Install using npm:

```sh
npm i @xcheme/cli
```

## Usage

Seeing all options available:

```sh
xcm --help
```

Generating a parser in JavaScript:

```sh
xcm -s ./path/to/source.xcm
```

Running a parser instantly:

```sh
xcm -s ./path/to/source.xcm -t ./path/to/file.ex --run
```
